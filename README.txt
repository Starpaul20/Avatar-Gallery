## INFORMATION ##

Avatar Gallery (1.5) for MyBB 1.8
Created by: Starpaul20
Copyright: ©2014
License: GPL

Adds an avatar gallery to your board. Based on the same feature that existed in MyBB 1.6 and earlier. Also allows administrators to manage the avatar gallery from the Admin CP, allowing them to upload, rename, move or delete avatars or galleries and update any users using them.

This plugin offers full language support.


## INSTALLATION ##

1. Upload all files above, keeping the file structure intact.
2. Go to Configuration > Plugins
3. Click "Activate"
4. Enjoy!


## UPDATING ##

If you're updating from any previous version, you must first deactivate the plugin, upload all new files and reactivate.
